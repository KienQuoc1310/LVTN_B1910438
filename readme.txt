## Hướng dẫn chạy dự án

1. Tạo cơ sở dữ liệu và đặt tên đúng với tên trong file cấu hình (.env)
2. Download composer 
3. Đổi lại cấu hình trong file .env và database.php sao cho phù hợp.
4. Mở console hoặc terminal tại thư mục gốc của dự án chạy những lệnh phía dưới
5. Chạy lệnh "composer install"
6. Chạy lệnh "php artisan key:generate"
7. Chạy lệnh "php artisan migrate"
8. Chạy lệnh "php artisan db:seed"
9. Chạy lệnh "php artisan serve"
10. Mở trình duyệt web và đi tới đường dẫn http://localhost:8000
